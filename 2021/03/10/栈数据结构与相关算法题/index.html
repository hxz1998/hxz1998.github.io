1. 用栈操作构建数组

来自力扣题库【1441】题：用栈操作构建数组

1.1 题目描述

给你一个目标数组 target 和一个整数 n。每次迭代，需要从 list = {1,2,3..., n} 中依序读取一个数字。

请使用下述操作来构建目标数组 target ：

  • Push：从 list 中读取一个新元素， 并将其推入数组中。
  • Pop：删除数组中的最后一个元素。
  • 如果目标数组构建完成，就停止读取更多元素。

题目数据保证目标数组严格递增，并且只包含 1 到 n 之间的数字。

请返回构建目标数组所用的操作序列。

题目数据保证答案是唯一的。

注意：

  • 1 <= target.length <= 100
  • 1 <= target[i] <= 100
  • 1 <= n <= 100
  • target 是严格递增的。

2. 删除字符串中的所有相邻重复项

来自力扣题库【1047】题：删除字符串中的所有相邻重复项

2.1 题目描述

给出由小写字母组成的字符串 S，重复项删除操作会选择两个相邻且相同的字母，并删除它们。

在 S 上反复执行重复项删除操作，直到无法继续删除。

在完成所有重复项删除操作后返回最终的字符串。答案保证唯一。

注意：

  1. 1 <= S.length <= 20000
  2. S 仅由小写英文字母组成。

2.3 思路

坦白地说，最一开始想到的不是栈，而是递归算法，还好悬崖勒马浪子回头，最后以栈方法来完成。

那么这道题是个简单题，具体思路就是用栈去保存要保留的字符序列，读进来的字符不断去跟栈顶元素相检查，如果一样就删掉，如果不一样就直接压进来。

3. 验证栈序列

来自力扣题库【946】题：验证栈序列

3.1 题目描述

给定 pushed 和 popped 两个序列，每个序列中的 值都不重复，只有当它们可能是在最初空栈上进行的推入 push 和弹出 pop 操作序列的结果时，返回 true；否则，返回 false 。

注意：

  1. 0 <= pushed.length == popped.length <= 1000
  2. 0 <= pushed[i], popped[i] < 1000
  3. pushed 是 popped 的排列。

3.2 示例数据

3.3 思路

使用栈去模拟整个过程，首先使用两个游标 i 和 ip 来代表 push 的游标和 pop 的游标。然后把 push 数组里面的数据挨个放到栈中，在放的过程中如果栈顶元素和 ip 指向的元素相同了，那么就直接弹出栈里面的所有符合前面条件元素。

栈相关的题目还是比较简单的，但有时候只是需要一个模拟栈的操作，并不是真的需要一个栈。所以优化时候要尽量往这里考虑。
